Diesel PBRT

Extends the PBRT API Framework

PBRT is an amazing open source API for Physical Based Rendering and Clean Interfaces Into the System. DieselPBRT is just an extension with few keys in mind.

  • API Class Context - Remove Static Inlined PBRT Render Process and allow stateful configuration
  • PBRT Thread Context - Intialize PBRT in its own threaded context / Provide Async Buffers We Can Sample From
  • SIMD Vector Operations - SIMD Operations provide significant speedup of Vector Operations
  • Diesel Export - PBRT primary Scene Descriptor - Primarily Export Descriptors and Handle for GPU configuration
